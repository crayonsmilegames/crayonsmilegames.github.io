# CrayonSmile GitHub Pages site

This folder is ready to publish as the free GitHub Pages site for the `crayonsmilegames` GitHub account.

Target public URL:

- `https://crayonsmilegames.github.io/`
- `https://crayonsmilegames.github.io/privacy/`

Recommended repository name:

- `crayonsmilegames.github.io`

Publishing steps:

1. Create a public GitHub repository named `crayonsmilegames.github.io`.
2. Upload all files from this folder to the repository root.
3. Commit to the `main` branch.
4. In GitHub, open Settings > Pages.
5. Set Source to `Deploy from a branch`, branch `main`, folder `/root`.
6. Wait until GitHub Pages shows the published URL.
